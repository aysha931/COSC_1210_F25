### ----- Game description ----- 
In this game, you are a goose. You must avoid balls and collect bread. As you collect bread, more balls appear! To win, you must collect all the bread.

### ----- How to run game  ----- 
- run notThatGooseGame.ipynb in vs code or other
- requires pygame

### ----- Controls   ----- 
- press the spacebar to start the game, ESC to end
- use up, down, left, right arrows to move (or w, s, a, d)
